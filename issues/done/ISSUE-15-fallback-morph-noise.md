# ISSUE-15: フォールバック形態素解析が極めてノイズが多い

## 深刻度
中

## 概要
Janome未インストール時のフォールバックとして `KANJI_KANA` 正規表現
`[一-龥ぁ-ゟ]+` を使っている。この正規表現は漢字・ひらがなの連続を1つのマッチとして
抽出するため、助詞や動詞を含んだ文全体が1つの「用語」として扱われる。

## 再現
```python
from explain_lint.patterns import KANJI_KANA
KANJI_KANA.findall("量子力学と固有値について議論する")
# → ['量子力学と固有値について議論する']  （文全体が1用語になる）
```

## 期待される挙所
最低限、助詞（と、に、つ、て、は、が、を、の等）で分割されるべき。
完全な形態素解析には及ばないが、実用に耐えるレベルの分割が必要。

## 影響範囲
- `patterns.py`: `KANJI_KANA` 正規表現
- `extract.py`: `_morph_terms()` のフォールバックパス

## 修正方針
- 助詞・接続詞を区切り文字として扱うよう正規表現を調整
  例: `[一-龥]+` と `[ぁ-ゟ]{2,}` に分割し、1文字のひらがな（助詞）を除外
- または `KANJI_KANA` を `[一-龥]+` のみに絞り、ひらがな連続は別途処理
